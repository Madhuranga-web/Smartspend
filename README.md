# SmartSpend AI 💰
AI-powered Personal Finance Tracker built with NestJS, Next.js, and Google Gemini AI.

## Features
- 📊 Track expenses with AI categorization.
- 🔐 Secure JWT Authentication.
- 📈 Visualized spending habits.
- 🤖 Gemini AI integration for financial insights.

## Tech Stack
- **Frontend:** Next.js, Tailwind CSS
- **Backend:** NestJS, MongoDB (Mongoose)
- **AI:** Google Gemini API
